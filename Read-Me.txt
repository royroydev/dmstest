1. Download the below files from github.
	a. dms.jar
	b. mysql-connector-java-8.0.11.jar
	c. tables.xml (Sample Files)
	d. param.properties (Sample file for Mysql, you can change the data based on your database configuration)
	e. running-success-image.jpeg
2. copy into your file system (Ex : C:\dms)
3. open the command prompt (Ex: C:\dms)
4. run below command 
	java -jar dms.jar

You can see the console data displayed about the process.

To see the xml generated insert below data or even more. and again run. You can see STudents data generated in student.xml in file system (C:\dms)
